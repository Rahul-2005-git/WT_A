# Array Operations

A simple JavaScript website demonstrating array operations: reverse, sorting, and searching.

## Operations

- **Reverse** - Reverses array elements
- **Sort Ascending** - Sorts from smallest to largest
- **Sort Descending** - Sorts from largest to smallest
- **Linear Search** - Sequential search (any array)
- **Binary Search** - Fast search (sorted array)

## Run

1. Double-click `index.html` OR
2. Use local server:
   ```bash
   python -m http.server 8000
   ```

## How to Use

1. Enter array numbers (comma-separated)
2. Click any operation button
3. See result displayed below
4. For search: enter value and click Search

## Files

- `index.html` - Main file
- `styles.css` - Styling
- `script.js` - Array operations

That's it!

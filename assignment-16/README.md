# Assignment 16 - Currency Converter (ReactJS)

## Features
- Convert **USD ↔ INR** in both directions
- Live exchange rate fetched from public API (fallback to approximate rate if offline)
- Quick amount buttons for common values
- Conversion history (last 10 conversions)
- React state + event handlers (useState, useEffect)
- No build step required — runs in browser directly

## How to Run

### Option 1: Open directly in browser (Recommended)
```bash
# Just open the file in browser:
open index.html
# OR on Linux:
xdg-open index.html
```

### Option 2: Serve with Node.js
```bash
npx serve .
# Then open http://localhost:3000
```

### Option 3: PHP server
```bash
php -S localhost:8016
# Then open http://localhost:8016
```

> **Note:** React is loaded via CDN. No npm install needed!

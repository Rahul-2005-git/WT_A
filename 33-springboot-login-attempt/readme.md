# Assignment 33 - Login Attempt Restriction

## 📌 Features
- Limits login attempts
- Blocks user after multiple failed attempts
- Secure authentication logic
- REST API-based login system

---

## ⚙️ Requirements
- Java 17+
- Maven 3.6+

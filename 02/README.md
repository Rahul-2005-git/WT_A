# VIT Projects - SDP, EDI, DT, Course

A simple interactive website showcasing the relevance of SDP, EDI, DT, and Course projects in VIT using Bootstrap tabs/pills and jQuery.

## Quick Start

1. **Open directly in browser:**
   - Double-click `index.html`

2. **Using local server:**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```
   Visit: `http://localhost:8000`

## Features

✅ **Interactive Tabs/Pills** - Click to switch between projects  
✅ **Responsive Design** - Works on all devices  
✅ **Smooth Animations** - Tab transitions and hover effects  
✅ **Comparison Table** - Quick side-by-side comparison  
✅ **jQuery Interactions** - Dynamic content loading  
✅ **Mobile-Friendly** - Optimized for tablets and phones

## What's Included

- **SDP** - Software Development Project details

## Browser Support

Chrome, Firefox, Safari, Edge (all latest versions)

## Dependencies (via CDN)

- Bootstrap 5.3.0
- jQuery 3.6.0
- Font Awesome 6.4.0

No installation needed - everything loads from CDN!

## Tips

- **Switch tabs:** Click pills or use arrow keys (← →)

- **Print:** Ctrl+P to print
- **Features:** Hover over badges and table rows for effects

# Assignment 20 - Tic-Tac-Toe Game (PHP)

## Features
- Classic 3×3 Tic-Tac-Toe game
- **Two modes:**
  - 👥 Player vs Player (same device)
  - 🤖 Player vs CPU (smart AI with win/block logic)
- Animated winner highlight with pulsing effect
- Winner overlay popup
- Score tracker (X wins / O wins / Draws) persisted in PHP session
- No database required — session-only game state
- Fully responsive design

## How to Run

```bash
php -S localhost:8020
```

Then open: http://localhost:8020

**No database setup needed!**

## How to Play
1. Choose mode: 2 Players or vs CPU
2. Click any empty cell to place your mark
3. First to get 3 in a row (horizontal, vertical, or diagonal) wins!
4. Click "New Game" to play again — scores are tracked

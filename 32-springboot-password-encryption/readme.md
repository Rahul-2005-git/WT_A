# Assignment 32 - Spring Boot Password Encryption

## 📌 Features
- Secure password hashing using BCrypt
- REST APIs for user registration
- H2 in-memory database
- Spring Security integration

---

## ⚙️ Requirements
- Java 17+
- Maven 3.6+

---

## 🚀 Setup & Run

### 1️⃣ Run application
mvn spring-boot:run

---

## 🌐 Access

- API: http://localhost:8080
- H2 Console: http://localhost:8080/h2-console

---

## 📂 Project Structure
src/main/java/
│── controller/
│── service/
│── model/
│── repository/

---

## ⚠️ Notes
- Use BCrypt for password encryption
- Configure H2 console credentials in application.properties

# Assignment 35 - Task Manager API

## 📌 Features
- CRUD operations for tasks
- REST API using Express.js
- Task status management
- JSON-based responses

---

## ⚙️ Requirements
- Node.js 18+
- npm

---

## 🚀 Setup & Run

### 1️⃣ Install dependencies
npm install

### 2️⃣ Run server
npm run dev

---

## 🌐 API Endpoint
http://localhost:3001/api/tasks

---

## 📂 Project Structure
│── routes/
│── controllers/
│── models/
│── server.js

---

## ⚠️ Notes
- Use Postman to test APIs

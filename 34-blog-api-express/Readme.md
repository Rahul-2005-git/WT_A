# Assignment 34 - Blog API (Express)

## 📌 Features
- REST API for blog management
- Create, read, update, delete blogs
- JSON-based API responses
- Express.js backend

---

## ⚙️ Requirements
- Node.js 18+
- npm

---

## 🚀 Setup & Run

### 1️⃣ Install dependencies
npm install

### 2️⃣ Run server
npm run dev

---

## 🌐 API Endpoint
http://localhost:3000/api/blogs

---

## 📂 Project Structure
│── routes/
│── controllers/
│── models/
│── server.js

---

## ⚠️ Notes
- Use Postman for API testing
- Ensure port 3000 is free
